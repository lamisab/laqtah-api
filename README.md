# Red3lay API

## Features
1. Login
2. Create contact
3. Update contact
4. Delete contact
5. View all contacts
6. Search contacts by name

## Stack
1. Runtime Environment: `Node.js`
2. Language: `TypeScript`
3. Web Server: `Fastify`
4. ORM: `Prisma`
5. DB: `Postgres`

## Requirements
1. At least 10 endpoints
2. Authentication (JWT)
3. Validation
4. No Supabase
5. Deployed to AWS or Heroku
